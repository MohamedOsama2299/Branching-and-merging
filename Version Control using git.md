# Command Comparison Reference

| Command        | Function                                               | Benefit                                          | Platform/Tool        |
| -------------- | ------------------------------------------------------ | ------------------------------------------------ | -------------------- |
| `touch`        | Create a new empty file                                | Quickly start a new file without content         | Mac/Linux            |
| `mkdir`        | Create a new folder                                    | Organize files into directories                  | Mac/Linux/Windows    |
| `dir`          | List folder contents                                   | See existing files and folders                   | Windows              |
| `New-Item`     | Create a file or folder                                | Alternative to `touch` and `mkdir` in PowerShell | Windows (PowerShell) |
| `open`         | Open a file or folder with default application         | Quickly run files from terminal                  | Mac/Linux            |
| `code`         | Open files/folders in VS Code                          | Speed up coding without manually opening VS Code | Mac/Linux/Windows    |
| `ls -a`        | List all files including hidden                        | See all files including hidden ones              | Mac/Linux            |
| `ls -Force`    | List all files including hidden/system                 | Check all files without exceptions               | Windows (PowerShell) |
| `git commit`   | Save changes with a message                            | Record project history in an organized way       | Git                  |
| `git log`      | Show commit history                                    | Track project changes and who made them          | Git                  |
| `git add .`    | Stage all changes for commit                           | Prepare changes to be saved in Git               | Git                  |
| `git checkout` | Switch branches or restore files                       | Test a new branch or revert changes              | Git                  |
| `git diff`     | Show differences between current files and last commit | Review changes before committing                 | Git                  |
